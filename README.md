Este projeto foi desenvolvido com o objetivo de facilitar o cálculo de números. A aplicação recebe os dados inseridos, processa a média aritmética utilizando JavaScript e exibe o resultado final de forma clara e rápida na interface.

## ⚙️ Funcionalidades do Sistema

O sistema foi estruturado para oferecer uma experiência simples e funcional:

* **Cálculo Automático:** Processamento da média realizado via script.
* **Exibição de Resultado:** Mostra na tela se o aluno foi aprovado ou reprovado com base na média.
* **Interface Limpa:** Design focado na facilidade de uso pelo usuário.

## 🚀 Tecnologias e Ferramentas

* **Linguagem:** JavaScript
* **Estrutura:** HTML5
* **Estilização:** CSS3
